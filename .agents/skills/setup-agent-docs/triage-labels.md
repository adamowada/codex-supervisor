# Triage Labels

Canonical roles:

- `needs-triage`: maintainer needs to evaluate.
- `needs-info`: waiting on reporter or HITL.
- `ready-for-agent`: AFK-ready with enough context and verification.
- `ready-for-human`: cannot be delegated yet.
- `wontfix`: deliberately not actioned.

Map these to the repo's actual labels or issue states. Do not create duplicate labels when an equivalent exists.
