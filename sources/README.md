# Sources

This directory contains shallow clones of OSS/public repositories used for inspiration and possible
integration experiments.

The cloned repositories are ignored by git. Keep this README tracked so future agents understand why
the directory exists.

## Cloned Repositories

- `openai-codex`: OpenAI Codex CLI and related source.
- `harnesslab-claw-code-agent`: Python coding-agent runtime and local-model reference.
- `openclaw-openclaw`: local-first agent control plane reference.
- `mattpocock-skills`: composable engineering/productivity skills.
- `mattpocock-sandcastle`: sandboxed agent orchestration and worktree patterns.
- `mattpocock-evalite`: TypeScript-native LLM eval tooling.
- `mattpocock-agent-rules-books`: agent rule packs derived from software engineering books.
- `mattpocock-agent-browser`: AI-friendly browser automation CLI.
- `mattpocock-node-DeepResearch`: iterative research/search loop reference.

## Use Rules

- Study architecture, tests, interfaces, and workflows.
- Respect licenses.
- Treat license-unclear repos as reference-only until reviewed.
- Do not copy source code into this project without a decision record.
